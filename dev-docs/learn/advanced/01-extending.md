# Advanced: Adding New Features

**Learning Goal**: Learn the complete workflow for extending Hei-DataHub with new functionality.

*(This page is under construction. Check back soon!)*

## Preview

Step-by-step guides for:

1. **Adding a new filter type**
   - Update query parser (Core)
   - Add to search service (Services)
   - Update autocomplete (UI)

2. **Adding a new badge**
   - Define badge logic (Core/Rules)
   - Render in UI (Widgets)
   - Style with TCSS

3. **Adding a new metadata field**
   - Update schema (Core/Models)
   - Update database (Infrastructure)
   - Add to edit form (UI)

4. **Adding a new view**
   - Create screen file (UI/Views)
   - Add navigation (Keybindings)
   - Write TCSS styling
