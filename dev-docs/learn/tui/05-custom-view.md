# Adding Your Own UI Screen

**Learning Goal**: Build a complete custom screen from scratch.

*(This page is under construction. Check back soon!)*

## Preview

This tutorial will walk you through creating a **Statistics Screen** that shows:
- Total number of datasets
- Most common tags
- Most active authors
- Recent activity timeline

You'll learn:
- Screen lifecycle (mount → compose → render → unmount)
- Loading data asynchronously
- Updating widgets in real-time
- Navigating between screens
- Passing data between screens
