## ‚ú® What's New

### New About Screen

A minimalistic, retro-inspired About screen accessible via `Ctrl+a`:

- **Project Story**: Learn about Hei-DataHub's origins and mission
- **Quick Links**: Direct access to GitHub repo and documentation
- **Retro Design**: Clean typography with HeiPlanet brand colors
- **Vim Navigation**: Full support for `j/k`, `gg/G`, and `d/u` scrolling

```bash
# Open from home screen
Press Ctrl+a
```

### Enhanced Search Filters

New metadata filters for more precise dataset discovery:

- **Source Filter**: `source:kaggle`, `source:github`, etc.
- **Format Filter**: `format:csv`, `format:parquet`, etc.
- **Tags Filter**: `tags:climate`, `tags:ml`, etc.

```bash
# Example searches
climate tags:weather format:csv
source:kaggle tags:nlp
```

### Improved Filter Badges

Visual enhancements to query representation:

- **Consistent Colors**: Hash-based color assignment for badge text
- **Better Layout**: Vertical overflow support, optimized padding
- **Retro Palette**: Curated colors matching the overall aesthetic

### Vim-Style Navigation

Enhanced keyboard shortcuts for power users:

- **`gg`**: Jump to top of list (first dataset)
- **`G`**: Jump to bottom of list (last dataset)
- **`Ctrl+A`**: Show About screen
- **`j/k`**: Navigate up/down in all scrollable areas

## üí° Developer Notes

### Migration Guide

If you were using the GitHub PR workflow or Outbox feature:

1. **Switch to WebDAV**: Use `hei-datahub auth setup` to configure cloud storage
2. **Direct Upload**: Datasets now upload directly to HeiBox on save
3. **No Outbox Needed**: WebDAV operations are synchronous with error notifications

### Code Organization

- All legacy GitHub/Git integration code has been removed
- Focus is now entirely on WebDAV cloud-native workflow
- Cleaner codebase with ~500 lines of unused code removed

### CLI Aliases

- `hei-datahub` is the primary command
- `mini-datahub` remains as a legacy alias (not deprecated yet)
  - Will be removed in a future release
  - Start migrating scripts to use `hei-datahub`
